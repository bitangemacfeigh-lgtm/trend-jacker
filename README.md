\# 🚀 Trend-Jacker: Social Media Scout



\*\*Trend-Jacker\*\* is an automated social listening and content generation tool. It monitors specific niches (like "AI in Kenya") for breakout trends and instantly generates high-engagement X (Twitter) threads and LinkedIn thought-leadership posts using the \*\*Mistral AI\*\* Large Language Model.



\---



\## ✨ Key Features

\* \*\*Real-time Monitoring:\*\* Tracks Google Trends for "Breakout" queries in specific geographical niches.

\* \*\*AI Content Engine:\*\* Powered by Mistral AI to draft viral-ready scripts.

\* \*\*Founder-Focused:\*\* Designed for brand builders looking for "organic dominance" without spending hours on research.

\* \*\*Deployment Ready:\*\* Optimized for Render with automated health checks and background-scouting capabilities.



\---



\## 🛠️ Tech Stack

\* \*\*Language:\*\* Python 3.10+

\* \*\*AI Model:\*\* Mistral-Large-Latest (via Mistral API)

\* \*\*Data Sources:\*\* Pytrends (Google Trends API)

\* \*\*Web Framework:\*\* Flask (for deployment \& manual triggers)

\* \*\*Server:\*\* Gunicorn
